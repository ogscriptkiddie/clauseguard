How It Works
The document is segmented into legal clauses using spaCy, then classified against 757 rules across 7 risk categories with negation detection. A weighted score from 0–100 is returned in milliseconds.

About the Project

Legal language
you're never meant

Built as a capstone security research project at Simon Fraser University. ClauseGuard treats legal document analysis as a defensive security problem — surfacing the risk language that platforms count on users never reading.
